# 更新记录

## 2026-05-27

- 内容类型新增“冲泡流程图”，支持冲泡步骤、器具、投茶量、水温、时间、箭头顺序和品饮结果的完整提示词
- 文生图支持粘贴完整文案后自动拆分封面文案和内页文案，生成封面时不会把内页文案写入画面
- 文生图模式新增文案用途判定，可自动判断封面文案或内页文案，并选择先生成封面提示词或内页提示词
- 封面文案自动归类为大字图或排名图：包含排名、榜单、PK、对比等表达时走排名图，否则走大字图
- 图生图模式新增“复制识图分析提示词”，可把图片发给 ChatGPT 后让其自动识别原图类型、平台、风格、主体和保留边界，再生成最终图生图提示词
- 导航页同步补充 GPT 识图分析提示词说明
- 原对比类内容类型改为“排名图”，联动结构改为从小到大、从大到小、金字塔、Top榜单、阶梯、表格、象限和横向排序
- 图生图“修改要求”新增常用预设选项，并保留自定义补充输入
- 图生图模式简化为一个“修改要求”输入框，系统自动补全保留原图风格和按平台/内容类型/文案生成新版图片的规则
- 未选择内容类型时不显示提示词，选择内容类型后才生成结果
- 提示词生成器拆分为“文生图 / 图生图”两个模式 Tab，图生图模式独立显示上传图片描述和改图目标
- 上传图片改图时会优先保留原图风格、色调、材质、光影和主体气质，再按平台、内容类型和文案生成新版画面
- 排名图新增专用排序结构，可选从小到大、从大到小、金字塔、Top榜单、阶梯、表格、象限和横向排序
- 所有下拉选项支持“不指定”，默认和重置状态不再强制绑定平台、内容类型、茶类、构图或风格预设
- 茶叶品类支持“自定义 / 不指定茶叶品类”，可由用户文案或自定义茶名决定茶叶对象
- 提示词生成器新增“是否写入文案”和“是否基于已有图片修改”判断
- 写入文案时可填写指定文字和排版位置，改图时可填写当前图片描述和修改目标
- 提示词生成器新增“发布平台”和“内容类型”两个独立维度
- 发布平台支持视频号、抖音、小红书、朋友圈、贴图，并自动写入画幅、阅读场景和文字密度判断
- 内容类型精简为知识图、产品图、产品详情图、生活图、大字图、排名图，避免平台用途和画面类型混在一起
- 导航页改为单屏工具面板，移除顶部说明区和底部说明区，页面本身不再上下滚动
- 导航页改为企业工具站视觉风格，弱化装饰感，强化工具入口与功能说明
- 移除导航页中的“查看项目代码”入口
- 优化 PC、平板、手机响应式布局：PC 双栏，平板横向工具分组，手机纵向可滚动工具面板
- 将导航页改为左侧工具列表、右侧功能详情的双栏结构
- 将导航首页扩展为多工具入口
- 新增 ChatGPT、Gemini、豆包、Claude、Kimi、DeepSeek、即梦 AI、Midjourney、LiblibAI 等外部工具跳转
- 新增导航首页 `index.html`
- 将提示词生成器页面迁移到 `prompt-generator.html`
- 将“空间层次”改为更通用的“画面结构”
- 增加信息图、产品图、意境图、茶空间、微缩景观、古典茶席等结构表达
- 调整内容类型逻辑：内容类型只表示画面形式，不再绑定具体茶叶品类
- 将“碧螺春茶叶知识图鉴”改为“茶叶知识图鉴 / 科普信息图”
- 将“祁门红茶知识图鉴”改为“茶叶工艺解析图 / 风味百科海报”
- 生成提示词时由“茶叶品类”统一决定具体茶名，避免同一提示词出现两个茶叶品类
- 初始整理为可部署静态站点
- 将入口文件整理为 `index.html`
- 添加部署说明和维护说明
- 工具支持 5 个发布平台、7 类内容类型、29 个茶叶品类、12 种构图、14 种风格质感
- 每个提示词模块都支持单独变更
